To raise project:

1. add to hosts file:
server-ip      mysite.loni

2. write in cmd
docker-compose build
docker-compose up -d

3. browse https://mysite.loni

to raise project without cancer:

1. add to hosts file:
server-ip      mysite.loni

2. in chat-backend:
  npm i
  npm run dev

3. in chat-frontend:
   npm i
   npm run start

4. make sure you have mongodb installed on pc (mongo compass recommended)
5. browse https://mysite.loni
